题目描述：

​ 有一个包含长度为N的整数数组nums，可正可负可零，现在我们要从中选取K个数，每个数之间的索引差不能超过D，怎样选择才能让选出来的数的乘积最大？请输出最大乘积的值，并且输出一个可能选择的组合（如果有多个组合，输出一个即可）。

示例：

* 输入：nums = [1, 3, 5, 2, 6],   K = 3,   D = 3   
* 输出：max = 90,    result = [6, 5, 3]
* 解释：6，5，3之间的索引值相差都不超过3，并且乘积最大，等于6X5X3 = 90

思路：
